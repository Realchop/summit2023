# Summit 2023 app

## Prerequisites
Node (https://nodejs.org/en)  
Ionic (npm i -g @ionic/cli) 

## Installation
git clone https://github.com/Realchop/summit2023.git  
cd summit2023  
npm i

## Running
### Web
ionic serve
### Android/iOS 
npx cap run android/ios  
To build, replace run/serve with build in previous commands.


### Web
ionic build
### Android 
npx cap run android
### IOS
npx cap run ios
